Accessibility Statement

Декларация о
Доступности.

Последнее обновление: 22 Апреля 2026

Компания DEHR Ltd считает, что интернет должен быть доступен всем. Мы стремимся обеспечить беспрепятственный доступ к нашему веб-сайту для людей с ограниченными возможностями в соответствии с Законом о равных правах людей с ограниченными возможностями (1998 г.) и правилами доступности услуг Израиля (2013 г.).

Уровень соответствия

Мы работаем над тем, чтобы наш сайт соответствовал стандарту WCAG 2.1 (Руководство по обеспечению доступности веб-контента) на уровне AA, насколько это технологически возможно. Сайт оптимизирован для работы со вспомогательными технологиями, поддерживает навигацию с клавиатуры и использует семантическую верстку.

Что сделано для доступности

  • Контрастность и читаемость: Цвета сайта подобраны с учетом необходимого контраста. Поддерживается темный и светлый режимы. Текст может быть масштабирован в браузере без потери функциональности.
  • Адаптивность: Сайт корректно отображается на любых устройствах (компьютеры, планшеты, смартфоны).
  • Навигация с клавиатуры: Все интерактивные элементы (кнопки, ссылки, поля форм) доступны для фокуса и использования с помощью клавиши Tab.
  • Семантика: Использованы правильные HTML-теги для заголовков, форм и списков.

Физическая доступность офиса

Наш офис, расположенный по адресу [ВАШ ПОЛНЫЙ АДРЕС ОФИСА], оборудован в соответствии с требованиями доступности:

  • Имеются парковочные места для инвалидов в непосредственной близости.
  • Вход в здание доступен для инвалидных колясок (пандусы/лифты).
  • В здании имеются доступные туалетные комнаты.
